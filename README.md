# Proyecto-de-ingeniera-de-datos-automatizaci-n-de-procesamientos-de-datos-del-INEGI-sobre-la-poblaci-
Se dasarrolla un pipeline etl en python que trabaja con un archivo csv sobre la población en Ciudad de México. Se implementan tecnologías como docker, airflow, tableau y postgresql para poder consultar, visualizar y reproducir dicho entorno.
